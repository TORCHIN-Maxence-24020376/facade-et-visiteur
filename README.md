# Les design patterns (Façade & Visiteur)
Ce dépôt GitHub contient le projet réalisé par un groupe de cinq étudiant(e)s en deuxième année de BUT Informatique.
Pour notre projet d'APP (= Apprentissage Par Projet), nous avons dû nécessairement nous informer, apprendre et coder les deux design patterns : la Façade et le Visiteur.

Ce projet a été exécuté dans un délai contraint de 8 heures.

Nous vous invitons donc à aller visiter la section `Wiki` de ce dépôt pour découvrir notre travail acharné sur les Façades et les Visiteurs.
